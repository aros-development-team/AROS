Class Compilation

Each topmost <class> element is compiled to a Javascript call to LzMakeClass

The <class> element is encoded as described in views.html (except that font metrics are not cascaded or computed), to produce an init object (initobj) that describes the initialization arguments, and the children, of the <class> element. 

Example: 

<class name="mywindow" extends="view">
 <property name="aroma" value="vanilla" />
 <attribute name="bgcolor" default="#FFFFFF" type="color" onset="this.applyWindowColor( bgcolor)" />
 <method name="mymethod" args="x,y,z">
  Debug.write("x is" + x + ", y * z =" + (y * z));</method>
  <text>MyWindow's Text</text> 
</class>
compiles to:
LzMakeClass({name: "mywindow", 
             attrs: {$setters: {bgcolor: function (bgcolor) {
this.applyWindowColor( bgcolor)},
                     name: "mywindow",
                     properties: {mymethod: function (x,y,z) {
                                                Debug.write("x is" + x + ", y * z =" + (y * z));
                                            }, 
                                  aroma: vanilla},
                     "extends": view},
            children: [{attrs: {label: "MyWindow's Text",
                                text: "MyWindow's Text"},
                        name: "text", 
                        children: []}]
  }, "view");
