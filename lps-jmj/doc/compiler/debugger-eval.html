Debugger Eval

Debug.eval(str) sends str in a request to the server in the format http://path/to/file.lzx?lzt=eval&lz_script=str, and plays the resulting movie.  The server compiles the string:

If it's an expression, it compiles a call to with (_root) Debug.write(str).

If it's a sequence of statements, it compiles code that executes with (_root) {str}.

Otherwise, it compiles a call to Debug.write("Parse Error: " + err), where err is the text of the error message that results from compiling str as a sequence of statements.
